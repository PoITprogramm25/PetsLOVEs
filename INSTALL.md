# Инструкция по установке Pet App

## Системные требования
- Node.js 16.x или выше
- npm 8.x или выше
- Expo CLI
- iOS Simulator (для разработки под iOS) или Android Studio (для Android)

## Шаг 1: Установка Node.js и npm

### Windows:
1. Скачайте Node.js с официального сайта: https://nodejs.org/
2. Установите, следуя инструкциям установщика
3. Проверьте установку:
```bash
node --version
npm --version
